## 7. 实际可用的探查方式

### 方式 A：使用项目 API（需要部署后登录）

**端点**: `GET /api/r2-files?recursive=true&prefix=showcase/`

**认证要求**:
- 需要 `requireAuth` 中间件（登录）
- 需要 `requireEditor` 角色

**使用步骤**:
1. 部署站点到 Cloudflare Pages
2. 使用 admin 账户登录
3. 访问上述 API 端点
4. 分析返回的文件列表，生成迁移映射

### 方式 B：配置 Cloudflare API Token（需要手动配置）

1. 在 Cloudflare Dashboard 创建 API Token（需要 User > Edit 权限）
2. 在本地 `.env` 添加：
   ```
   CLOUDFLARE_API_TOKEN=your_token_here
   ```
3. 使用 curl 调用 REST API：
   ```bash
   curl -X GET "https://api.cloudflare.com/client/v4/r2/buckets/lovelymain/objects?prefix=showcase/" \
     -H "Authorization: Bearer $CLOUDFLARE_API_TOKEN" \
     -H "Content-Type: application/json"
   ```
